# ⚙️ Setup & Configuration

This folder contains all guides needed to get your environment ready for GW research.

| File | Description |
|---|---|
| [miniconda_cluster.md](miniconda_cluster.md) | Installing Miniconda on a cluster — first time setup |
| [set_up.md](set_up.md) | MacOS environment, Miniconda, CVMFS, and IGWN conda environment |
| [lvk_account.md](lvk_account.md) | LVK account creation, LDG cluster access, Kerberos, SSH login |
| [bash_intro.md](bash_intro.md) | Introduction to Bash scripting for beginners |

---

## Recommended Order

1. **New to clusters?** → `miniconda_cluster.md` — install Python on the cluster first
2. **New to the group?** → `set_up.md` — configure your local machine
3. **Need cluster access?** → `lvk_account.md` — LVK account and LDG login
4. **Want to clone private repos on the cluster?** → `README_SSH_Cluster_Git_Access.md`
5. **Unfamiliar with the terminal?** → `bash_intro.md` — read this before anything else
